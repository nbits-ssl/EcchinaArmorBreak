## 概要

軽装、重装、衣服、宝石(これだけ設定可)のキーワードを持つ装備を、ブロックしていない攻撃
ごとに確率(設定可)で脱衣/破壊(設定可)します。
デフォルトでは100%の確率で宝石類も含む装備中のものを脱衣させます。動作確認後、お好みの
設定にしてください。(確率100%ですが、連続Hit時のため、0.5秒のウェイトがあります)


## 必要なもの

* SKSE


## アンインストール

  stopquest ecchinaarmorbreak

してからアンインストールしてください。


## 設定

MCMはめんどいので全部グローバル変数制御です。
コンソールで「set 変数名 to 値」で変更して下さい。

EABBreakMode (デフォルト1)

0 装備を破壊します
1 装備を脱衣します

EABEnableJewelry (デフォルト1)

0 宝石類が選択されたらパスします
1 宝石類も破壊/脱衣します

EABRate (デフォルト100)

破壊/脱衣の確率です
